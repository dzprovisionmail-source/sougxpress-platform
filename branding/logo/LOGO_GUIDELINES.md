# Logo Guidelines

## Official Status
Only SVG vector files located in `logo/approved/` are considered official production logos.

## File Usage
- **SVG**: Primary format for all digital and print production.
- **PNG**: Exports only for specific use cases where SVG is not supported.
- **Monochrome**: To be used when color reproduction is limited.
- **Horizontal/Vertical**: Specific lockups for different layout constraints.
- **Favicon**: Optimized versions for web browsers and small UI elements.

## Integrity
The logo must never be stretched, distorted, or modified without following the official evolution process.

# Soug-XPRESS V2 — Design System Architecture

## Overview
This directory contains the permanent, single source of truth for the Soug-XPRESS V2 visual language. This design system is built to govern every current and future interface of the platform, from the **Founder Operating System** to the **Customer Application**.

## Design Philosophy
Soug-XPRESS is an **Urban Commerce Operating System**. Its design is not merely aesthetic; it is a **governance interface**. 
- **Arabic-First (RTL)**: The primary language is Arabic, with built-in support for future bilingual (Arabic/French) expansion.
- **Mobile-First & Dark-Mode First**: Optimized for modern mobile usage and energy efficiency.
- **Component-Driven & Token-Based**: Built for scalability and consistency across React Native (Expo) and Web platforms.
